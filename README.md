# Aircraft Integration Kit

Модульный инструмент для интеграции данных о воздушных судах через внешние API, локальные файлы и внутреннюю логику работы с ВС.

## Структура проекта

- `api/` — модуль взаимодействия с REST API (получение данных о рейсах, статусах ВС)
- `file_handler/` — модуль чтения/записи файлов (CSV, JSON, Excel)
- `aircraft/` — модуль бизнес-логики для работы с воздушными судами (расчёт остатка топлива, проверка бортовых номеров и т.д.)

## Быстрый старт

1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать окружение и установить зависимости:  
   `pip install -r requirements.txt`
4. Настроить переменные окружения (скопировать `.env.example` в `.env` и заполнить)
5. Запустить: `python main.py`
