# ✈️ Aircraft Tracker

Консольное приложение для получения информации о самолётах над заданной страной через **OpenSky Network** и **Nominatim**.

## Возможности

- Ввод названия страны (на английском)
- Получение реального списка самолётов в воздушном пространстве
- Отображение количества найденных бортов
- Автоматическое сохранение результата в JSON после каждого запроса
  - `data/latest.json` — последний полученный результат
  - `data/<country>_YYYYMMDD_HHMMSS.json` — история снимков
- Меню с заготовками:
  1. Топ N по высоте
  2. Фильтр по стране регистрации
  3. Фильтр по диапазону высот
  4. Выход

> **Важно:** фильтрация и аналитика в меню пока в разработке.

## Установка и запуск

```bash
git clone <url-репозитория>
cd aero-handler
pip install requests
python main.py
```
Структура проекта

**Используемые API**<br>
_Nominatim_ – bounding box страны<br>
_OpenSky_ _Network_ – данные о самолётах

**ООП и SOLID**<br>
Абстрактные классы BaseAPI, BaseStorage – наследование OpenSkyAPI

Инкапсуляция в Aircraft (приватные атрибуты, геттеры, валидация)

Single Responsibility, Open/Closed – легко добавить другие API или хранилища

Планы по доработке
Полноценная фильтрация и сортировка в меню

Тесты (pytest)

Обработка ошибок сети и ввода
